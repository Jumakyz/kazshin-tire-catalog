# Final Report Text Draft

## Project Title
KazShin Tire Catalog and Customer Request Management Website

## Project Description
During my Industrial Practice, I developed a web-based tire catalog and customer request management website for KazShin, a tire sales company. The purpose of the project was to create a digital solution that helps customers browse available tire products, filter them by technical characteristics, and submit online requests for consultation or purchase.

## Technologies Used
The project was implemented using HTML, CSS, JavaScript, localStorage, Git, and GitHub. HTML was used to create the structure of the website. CSS was used for responsive design, layout, product cards, buttons, forms, and visual styling. JavaScript was used to implement the dynamic catalog, filtering, sorting, product selection, request form validation, saving requests in localStorage, and exporting request data to CSV.

## Technical Contribution
My technical contribution included designing and implementing the website interface, creating a dynamic tire catalog, adding filters by season, brand, rim size, and availability, implementing search functionality, creating a product detail modal, developing a customer request form, validating form data, saving customer requests locally, and implementing CSV export for request data.

## Result
The final result is a working website prototype that demonstrates how KazShin can present tire products digitally and collect customer inquiries in a structured format. The project includes practical evidence such as source code, GitHub commits, screenshots, and a published GitHub Pages link.
